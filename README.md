\# AI Resume Optimizer API \& Dashboard



A Full-Stack AI application that extracts text from a binary PDF resume upload, performs an automated gap analysis against a provided Job Description, and generates detailed ATS structured metrics using Google Gemini 2.5 Flash.



\## 🛠️ Tech Stack

\- \*\*Backend:\*\* FastAPI, Pydantic, Python

\- \*\*AI Integration:\*\* Google GenAI SDK (Gemini 2.5 Flash)

\- \*\*Parsing Engine:\*\* pdfplumber

\- \*\*Frontend:\*\* HTML5, Tailwind CSS, JavaScript (Fetch API)
